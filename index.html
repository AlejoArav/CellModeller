Welcome to Cellmodeller 4.

Cellmodeller is a python framework for multicellular modelling developed by Tim Rudge, PJ Steiner and Jim Haseloff at Cambridge University. The project evolved from work on plant cellular development by Tim Rudge, Lionel Dupuy and Jonathan MacKenzie in Jim Haseloff's lab from 2002-2008. Cellmodeller is intended to be a modular and extensible generic framework for multicellular systems. The current version includea GPU-based biophysical model of bacterial growth and from a single initial cell can generate > 100,000 cells in around 2 hours. Cell behaviours and dynamics are encoded as user-defined rules (written in python) and/or differential equation (written in OpenCL)

Check out the documentation at Cellmodeller Documentation

More details of the research and publications associated with Cellmodeller can be found at www.haseloff-lab.org
